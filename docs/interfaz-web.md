# Interfaz Web

La app unificada en Streamlit permite explorar los tres módulos desde una sola interfaz.

## Funciones

- Ver predicciones de demanda BUS y METRO.
- Subir una imagen y clasificar distracción al volante.
- Probar recomendaciones personalizadas por usuario o por filtros de catálogo.
- Consultar el informe resumido de distracciones y medidas preventivas.
